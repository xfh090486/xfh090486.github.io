1.访问时点击链接变为下载

原因是 _config.yml 中 permalink配置文件少了最后面的/
正确: permalink: :category/:title/

一些配置

  • 开启目录 toc: true
  • 开启评论 comments: true 要装评论插件
  • 折叠显示 使用 <!-- more --> 标记就会显示查看更多
  • 画廊 gallery: true

使用命令

  • hexo clean

  • hexo generate / hexo g

  • hexo deploy / hexo d

  • hexo server / hexo s

  • hexo new “My New Post”

  • hexo new page –path about/me “About me”

  • hexo clean && hexo g && hexo s
